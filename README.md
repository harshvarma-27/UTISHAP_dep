# UTISHAP

### Urinary Tract Infection Sanitary Health Alert Patch

UTISHAP is a wearable diagnostic system designed for early detection of urinary tract infections (UTIs) using a combination of **chemical biosensing and AI-based analysis**. The system integrates a smart diagnostic patch into a sanitary pad to enable **passive, real-time health monitoring**.

---

## Core Technology

UTISHAP is built on the integration of three key technologies:

### 1. Paper-Based Microfluidic Biosensing

* Uses microfluidic channels to guide urine flow across sensing zones
* Operates without electricity using capillary action
* Enables controlled and uniform distribution of fluid

---

### 2. Multi-Analyte Chemical Detection

The patch contains three chemically isolated sensing pads:

* **Nitrite Sensor**
  Detects bacterial activity associated with UTIs

* **Leukocyte Sensor**
  Identifies presence of white blood cells indicating infection

* **pH Sensor**
  Measures acidity/alkalinity to support diagnosis

Each pad produces a **colorimetric reaction** within 1–5 minutes.

---

### 3. AI-Based Digital Interpretation

* Mobile app captures the image of the patch
* Image processing extracts color intensity from each pad
* AI model analyzes patterns and predicts infection risk
* Eliminates human error in manual color interpretation

---

## System Workflow

### Step 1: Integration into Wearable

The diagnostic patch is embedded within a sanitary pad in the urine-contact region.

---

### Step 2: Passive Fluid Flow

During usage:

* Urine is absorbed into the pad
* Microfluidic channels guide fluid to sensing zones
* Each zone receives controlled exposure

---

### Step 3: Chemical Reaction

* Nitrite, leukocyte, and pH pads react independently
* Each produces a visible color change
* Reaction stabilizes without external power

---

### Step 4: Image Capture

* User removes the patch
* Scans it using a mobile application
* Image is preprocessed for analysis

---

### Step 5: AI-Based Analysis

* Color values are extracted from each segment
* Features are mapped to infection indicators
* Model generates a **risk score**

---

### Step 6: Output and Feedback

* Displays result: Normal / Possible Infection / High Risk
* Provides health suggestions and recommendations
* Stores historical data for tracking

---
